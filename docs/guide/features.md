# 功能概览

## 开发环境管理

面向多语言开发场景，集中管理运行时安装路径与当前生效版本：

- **支持语言**：Java、Node.js、Python、PHP、C++ 等（以实际配置为准）
- **版本切换**：多版本共存时快速切换，减少 PATH 冲突
- **路径管理**：配合 `BaseEnvPath` 统一存放环境文件

## 系统工具

按类别整理常用 Windows 工具，减少在「运行」或控制面板中查找：

| 分类 | 示例 |
|------|------|
| 系统管理 | 计算机管理、服务、系统配置、注册表、环境变量 |
| 硬件与设备 | 设备管理器、磁盘管理、显示/声音/电源 |
| 网络与安全 | 防火墙、本地安全策略、网络连接 |
| 性能与监控 | 任务管理器、资源监视器、事件查看器、计划任务 |

## 应用程序管理

- **本地应用**：将 `.exe` **拖入**程序即可保存路径与图标；支持多选添加、卡片单击启动
- **文件夹**：拖入目录作为快捷方式，可设**别名**与自定义图标，单击在资源管理器中打开（不重命名真实文件夹）
- **网站**：输入网址自动获取标题与 favicon，可改显示标题和图标，单击在浏览器打开
- **无边框窗口 + 托盘**：无系统标题栏；右上角仅关闭按钮（隐藏到托盘）；托盘双击或菜单可恢复窗口，菜单「退出」才真正结束程序
- **快捷操作**：运行、在资源管理器中显示、移除应用

## 服务管理

对本地开发依赖的服务进行启停或快捷访问（具体能力取决于本机已安装组件），例如：

- 数据库：MySQL、Redis、MongoDB
- Web：Nginx、Apache
- 消息：RabbitMQ、Kafka 等

## 其他

- 系统托盘与可选开机自启
- 浅色 / 深色主题（见 [配置说明](./configuration.md)）
- 状态栏显示当前版本（与 Release 标签一致）
